# Datastraw Support CRM

A full-stack Customer Support Ticketing CRM System built for the Datastraw hiring assessment. 

## Tech Stack
* **Backend:** Python, FastAPI, SQLite
* **Frontend:** Vanilla HTML, JavaScript, Tailwind CSS

## Local Setup Instructions
1. Clone this repository.
2. Install the requirements: `pip install -r requirements.txt`
3. Run the backend server: `python -m uvicorn main:app --reload`
4. Open `frontend/index.html` in your web browser.

## Core Features
* Create new support tickets
* View all tickets in a dashboard
* View detailed ticket information
* Update ticket status (Open, In Progress, Closed)
* Search by customer name and filter by status
